# Art Gallery Microservices

## Цель проекта
Создать микросервисную платформу для размещения и продажи произведений искусства (картин, фото, видео).

## Бизнес-ценность
- Образовательная: изучение современных технологий (Kafka, Docker, микросервисы, CI/CD).
- Портфолио: демонстрация архитектурного мышления и владения стеком.
- Возможный запуск как реального сервиса.

## Ключевые требования
- Масштабируемость: добавление новых сущностей (видео, отзывы, продажи) без переписывания кода.
- Асинхронная обработка через Kafka.
- Отказоустойчивость и наблюдаемость (логи, метрики).

## Стек
- Backend: .NET 8, ASP.NET Core
- Брокер: Kafka (Confluent)
- База: PostgreSQL (каждый сервис имеет свою схему)
- Gateway: YARP
- Frontend: React + TypeScript
- Оркестрация: Docker Compose, Kubernetes (позже)
- CI/CD: GitHub Actions
